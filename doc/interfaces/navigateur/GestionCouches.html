enleverDeclencheur(type, id, action)

Retirer un ou des Declencheurs.
Si pas de type, tous les Declencheurs de l'objet seront supprimés.
Si seulement le type, tous les Declencheurs de ce type seront supprimés.
Si le type et l'id, tous les Declencheurs de ce type ayant l'id seront supprimés.
Si le type et l'action, tous les Declencheurs de ce type ayant l'action seront supprimés.
